# Code of Conduct

The Amore project adopts the **Contributor Covenant 2.1** verbatim as its code of conduct.

The canonical text is published at:
https://www.contributor-covenant.org/version/2/1/code_of_conduct/

By participating in this project — by opening issues, submitting PRs, or
communicating on any official channel — you agree to abide by the Covenant.

## Scope

Applies to all project-controlled spaces: this repository, its issues, its
pull requests, its discussions, and any official chat / forum / mailing
list spawned by the project.

## Enforcement contact

Reports of unacceptable behaviour go to one of:

- `security@amore.dev` (subject prefix `[AMORE-COC]`) — placeholder until
  the mailbox is live at v1.0.0.
- `antonioakiki15@gmail.com` (subject prefix `[AMORE-COC]`) — fallback
  while the project mailbox is being set up.

Reports are reviewed within 5 business days. Reporter identity is
confidential by default. Retaliation against reporters is itself a
Covenant violation and treated as such.

## Enforcement guidelines

The Covenant ships with a 4-tier enforcement ladder
(Correction / Warning / Temporary Ban / Permanent Ban). Amore follows
that ladder. Maintainer judgement applies the ladder; appeals go to the
same contact address.

## Why a reference, not a copy?

The Contributor Covenant is a living document — version 2.1 is the
current revision but later revisions may land. Linking to the canonical
URL keeps Amore aligned with the upstream community standard.

If `contributor-covenant.org` ever goes offline, the Covenant 2.1 text
is mirrored in the Internet Archive Wayback Machine and on the project's
GitHub repository at `EthicalSource/contributor_covenant`.
