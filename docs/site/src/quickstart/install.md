# Install in 60 seconds

Pick the method that matches your OS. All install the same signed binary from the
[GitHub Release](https://github.com/antonio-amore-akiki/amore/releases).

## macOS / Linux — Homebrew

```bash
brew install antonio-amore-akiki/tap/amore
```

## Windows — winget

```powershell
winget install AntonioAmoreAkiki.Amore
```

## Linux — AUR (Arch)

```bash
yay -S amore-bin
```

## After install

Verify the binary:

```bash
amore --version
```
