# Support — Amore

Single-author OSS project. No commercial support tier. Three channels:

## 1. GitHub Issues

Bug reports + feature requests:
`github.com/antonio-amore-akiki/amore/issues/new/choose`

Templates available: Bug / Feature / Security.

## 2. Security disclosures

Use GitHub Security Advisories (private channel):
`github.com/antonio-amore-akiki/amore/security/advisories/new`

SLA: 5-day response per SECURITY.md.

## 3. Discussions

General questions + usage help:
`github.com/antonio-amore-akiki/amore/discussions`

---

## Response time

Best-effort. Single maintainer. No SLA except security (5-day per SECURITY.md).

For deployments requiring guaranteed SLA, self-hosting with an internal support team is the path — Apache-2.0 permits unrestricted commercial use and modification.
