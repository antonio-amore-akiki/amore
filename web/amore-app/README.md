# amore-app

Phase α MVP — Next.js 15 web front-end for [amore.app](https://amore.app).

## Dev setup

```bash
npm install
npm run dev
# open http://localhost:3000
```

## Deploy

Deployment is automatic via Vercel on every push to `main`. Link the repo at vercel.com once; subsequent pushes auto-deploy.

Build locally to verify before push:

```bash
npm run build
```
